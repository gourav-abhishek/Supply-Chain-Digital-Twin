# AI Predictive Supply Chain Digital Twin

A modern, web application that demonstrates a **AI Predictive Supply Chain Digital Twin** for Indian logistics.  
It visualizes an end-to-end supply chain, simulates scenarios, and uses a ML model for demand prediction and risk scoring.

## üöÄ Features

- **Digital Twin Visualization**
  - Interactive network of suppliers, warehouses, hubs, and retailers across Indian cities (Mumbai, Delhi, Bengaluru, Chennai, Pune, Nagpur, etc.).
  - Animated flows showing movement of goods between nodes.
  - Node-level KPIs: capacity, utilization, on-time rate, risk score.


- **Scenario Simulation**
  - Simulate ‚Äúwhat-if‚Äù scenarios:
    - Demand surge in specific regions (e.g., Mumbai, Delhi).
    - Port/warehouse disruption (e.g., Nhava Sheva, Chennai port).
    - Transport strike on specific routes.
  - Recomputes KPIs, demand, risks, and shows impact visually.

- **Dashboards & Analytics**
  - Real-time style dashboard cards (Shipments in transit, At-risk shipments, Average delay, Service level, etc.).
  - Time-series charts for historical vs predicted demand.
  - Route-level analytics: distance, time, cost, on-time %.
  - Inventory view: current stock, safety stock, reorder alerts.
